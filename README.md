setting-vim
===========

vim settings


##Usage
Setup Neobundle  
$ cd ~  
$ mkdir .bundle  
$ git clone https://github.com/Shougo/neobundle.vim.git .bundle/neobundle.vim  

Setup .vimrc  
$ cd ~  
$ curl -O https://raw.githubusercontent.com/akiyoko/setting-vim/master/.vimrc  

Setup Neobundle plugins  
$ vim  
$ :NeoBundleInstall
